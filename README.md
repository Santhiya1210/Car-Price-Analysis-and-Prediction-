# Car-Price-Analysis-and-Prediction

# Objectives

Analyze overall car price distribution and trends. Study how price varies based on mileage, year, engine volume, and car type. Identify patterns affecting car prices. Provide insights to understand factors influencing car value.

# Tools Used

* Python – Data analysis and preprocessing
* Pandas, NumPy – Data handling
* Matplotlib, Seaborn – Visualization
* Power BI – Dashboard creation

# Data Modeling

* Single structured dataset used for analysis.
* Derived features like car_age and drive_num created.
* Price considered as the target variable.

# Key Metrics

* Total Cars: 8838
* Average Price: 9.13
* Max Price: 12.61
* Min Price: 5.56
* Average Mileage: 142.82

# Key Insights

* Car price decreases with higher mileage and older cars.
* Engine volume has a positive impact on price.
* Sedan is the most common car type.
* Petrol cars dominate the dataset.
* Front-wheel drive cars are the most used.

# Conclusion

The analysis shows that car price is mainly influenced by mileage, age, and engine size.
Newer cars with lower mileage and higher engine capacity tend to have higher prices.
These insights help in better pricing and decision-making.
